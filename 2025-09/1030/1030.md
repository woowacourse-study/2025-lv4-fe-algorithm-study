# 📝 알고리즘 문제 풀이 - 2025-10-30

## 문제 📖

- **문제 링크:** [백준 - 신입사원](https://www.acmicpc.net/problem/1946)

- **문제 유형:** 정렬

- **난이도:** 실1

- **풀이 시간:** ⏳ 30분

## ✍ 접근 방식

- 다른 모든 지원자와 비교했을 때 서류심사 성적과 면접시험 성적 중 적어도 하나가 다른 지원자보다 떨어지지 않는 자
- 순위가 모두 떨어지면 안 뽑음
- 첫번째 요소(서류심사 성적 순위)를 기준으로 오름차순 정렬

  - [ 1, 4 ], [ 2, 5 ], [3, 6], [4, 2], [5, 7], [6, 1], [7, 3];

- 첫번째 요소는 정렬되어 있으므로 두번째 요소만 비교하면 된다
- 0번째 인덱스 = 1등, 1번째 인덱스 = 2등 ...
- 1등과 2등의 두번째 요소(면접 성적 순위)를 비교해서 2등이 더 낮다면 1등에 비해 모두 떨어지는 것이므로 제외한다.

  - [ 1, 4 ], [ 2, 5 ]

- 2, 5가 1, 4 보다 순위가 모두 낮으므로 2, 5를 제외
- 1, 4와 3, 6을 비교하면 3, 6이 순위가 모두 낮으므로 3, 6을 제외
- 1, 4와 4, 2를 비교하면 4, 2의 2가 1, 4의 4보다 순위가 높으므로 기준점을 4, 2로 변경
- 4, 2와 5, 7을 비교하면 5, 7이 순위가 모두 낮으므로 5, 7을 제외
- 반복
