## ✍ 접근 방식

> 어떤 방법으로 접근했는지 설명
> 다른 풀이를 참고했다면 어떤걸 배웠는지

- new Map();
- mapVariable

  - .has(key), .get(key), .set(key, value)

- 아래와 같이 배열로 받는 방법도 있다

```js
function solution(strArr) {
  const counter = Array(31).fill(0);
  for (const str of strArr) {
    counter[str.length]++;
  }
  return Math.max(...counter);
}
```
