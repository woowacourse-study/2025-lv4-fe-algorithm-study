# 📝 알고리즘 문제 풀이 - 2025-09-13

## 문제 📖

- **문제 링크:** [백준 - 줄 세우기](https://www.acmicpc.net/problem/2252)

- **문제 유형:** 위상 정렬

- **난이도:** 골드 3

- **풀이 시간:** ⏳ 1시간 20분

## 시간 복잡도: 최대 O(N + M)

- N: 학생 수, 노드 수
- M: 관계 수, 간선 수

## 공간 복잡도: O(N + M)

- arr: O(N + M) => 인접 리스트로, N + 1개의 배열 + 총 M개의 간선 정보
- inDegrees: O(N)
- queue: 최대 O(N)
- answer: O(N)
