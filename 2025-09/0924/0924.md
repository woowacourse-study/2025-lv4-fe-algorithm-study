## ✍ 접근 방식

> 어떤 방법으로 접근했는지 설명
> 다른 풀이를 참고했다면 어떤걸 배웠는지

### 깨달은거

arr.pop() === arr.slice(-1)

### 초기

```js
function solution(arr) {
  var stk = [];
  for (let i = 0; i < arr.length; i++) {
    if (stk.length === 0) {
      stk.push(arr[i]);
    } else {
      if (stk[stk.length - 1] === arr[i]) {
        stk.pop();
      } else {
        stk.push(arr[i]);
      }
    }
  }
  return stk.length > 0 ? stk : [-1];
}
```
