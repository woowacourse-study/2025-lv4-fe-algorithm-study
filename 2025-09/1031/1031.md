# 📝 알고리즘 문제 풀이 - 2025-10-31

## 문제 📖

- **문제 링크:** [백준 - 숨바꼭질3](https://www.acmicpc.net/problem/13549)

- **문제 유형:** dfs/bfs

- **난이도:** 골3

- **풀이 시간:** ⏳ 15분

## ✍ 접근 방식

- 가장 빠른 시간이 몇 초 후? = 최단거리 구하는 문제 -> bfs

- 0초 후 2\*x
- 1초 후 x-1, x+1
- nx = x+dx
- 만약 2\*x 면 visited[nx] = visited[x]
- x-1, x+1 이면 visited[nx] = visited[x]+1
- 시작이 1이고 도착이 2. -> x+1 , 2\*x 둘 다 갈 수 있다
- 더 작은 값 넣을 수 있어야함
