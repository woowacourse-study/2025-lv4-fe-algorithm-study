## ✍ 접근 방식

> 어떤 방법으로 접근했는지 설명
> 다른 풀이를 참고했다면 어떤걸 배웠는지

```js
const solution = (arr, k) => {
  const set = new Set(arr);
  const result = [...set].slice(0, k);
  while (result.length !== k) {
    result.push(-1);
  }
  return result;
};
```

```js
function solution(arr, k) {
  const set = new Set(arr);
  return set.size < k ? [...set, ...Array(k - set.size).fill(-1)] : [...set].slice(0, k);
}
```
