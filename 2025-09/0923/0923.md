# 📝 알고리즘 문제 풀이 - 2025-09-23

## 문제 📖

- **문제 링크:** [백준 - 바이러스](https://www.acmicpc.net/problem/2606)

- **문제 유형:** BFS, DFS, 그래프 탐색

- **난이도:** 실버3

- **풀이 시간:** ⏳ 20분

## ✍ 접근 방식

### DFS

- 시간 복잡도: O(N + E)
- 공간 복잡도: O(N + E)

### BFS - shift 방식

- 시간 복잡도: O(N^2 + E)
- 공간 복잡도: O(N + E)

### BFS - dequeue 방식

- 시간 복잡도: O(N + E)
- 공간 복잡도: O(N + E)
