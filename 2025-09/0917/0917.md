# 📝 알고리즘 문제 풀이 - 2025-09-17

## 문제 📖

- **문제 링크:** [백준 - 옥상 정원 꾸미기](https://www.acmicpc.net/problem/6198)

- **문제 유형:** 스택

- **난이도:** 골드5

- **풀이 시간:** ⏳ 14분

## ✍ 접근 방식

- stack에 빌딩 순차적으로 넣기
  - stack과 현재 빌딩 비교
    - stack의 top이 현재 빌딩 보다 더 크면 오른쪽 볼 수 있음
      - pop할 요소 없음
    - stack의 top이 현재 빌딩 보다 작거나 같으면 오른쪽 못 봄
      - stack의 top이 더 큰 높이가 될 때까지 pop
  - stack의 길이만큼 수를 더한다.
  - stack에 현재 빌딩의 높이를 넣는다.
