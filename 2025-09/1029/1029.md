# 📝 알고리즘 문제 풀이 - 2025-10-29

## 문제 📖

- **문제 링크:** [LeetCode - MoveZeroes](https://leetcode.com/problems/move-zeroes/)

- **문제 유형:** 투 포인터

- **난이도:** easy

- **풀이 시간:** ⏳ 19분

## ✍ 접근 방식

- 정수 배열 nums가 주어졌을 때, 0인 모든 요소를 배열의 끝으로 이동시키되, 0이 아닌 요소들의 상대적 순서는 유지해야 합니다.

- 배열의 복사본을 만들지 않고, 원본 배열을 직접 수정하는 방식으로 이 작업을 수행해야 합니다.
- 0을 뒤로 옮기려면 수를 비교 해야 하므로 투 포인터를 사용해야 한다고 생각했다.
- 수가 2개이므로 나올 수 있는 경우의 수는 4가지
  - 0인 경우, 0인 경우
  - 0인 경우, 0이 아닌 경우
  - 0이 아닌 경우, 0인 경우
  - 0이 아닌 경우, 0이 아닌 경우
