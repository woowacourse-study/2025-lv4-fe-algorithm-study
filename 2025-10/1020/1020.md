# 📝 알고리즘 문제 풀이 - 2025-10-20

## 문제 📖

- **문제 링크:** [백준 - 아기 상어 2](https://www.acmicpc.net/problem/17086)

- **문제 유형:** 그래프, 브루트포스

- **난이도:** 실버 2

- **풀이 시간:** ⏳ 25분

## ✍ 접근 방식

- 모든 상어의 좌표를 저장하여 큐에 넣은 뒤, 상하좌우 / 대각선의 0인 칸과의 거리를 채워가는 방식
  - 예에전에 풀었던 문제 다시 푼건데, 이 접근 방법이 안 떠올라서 기존 코드 살짝 염탐함 (왜 더 퇴화했냐 ..?)
- 배열의 최대값 - 1이 정답
