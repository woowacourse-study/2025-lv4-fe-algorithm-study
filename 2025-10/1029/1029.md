# 📝 알고리즘 문제 풀이 - 2025-10-29

## 문제 📖

- **문제 링크:** [백준 - 양](https://www.acmicpc.net/problem/3184)

- **문제 유형:** 그래프

- **난이도:** 실버 1

- **풀이 시간:** ⏳ 13분

## ✍ 접근 방식

- bfs
- 양 또는 늑대를 만나면 bfs 현재 좌표에서 시작
- 벽을 만나기 전이나 영역 밖으로 벗어나기 전까지 검사
- 현재 영역의 늑대의 수와 양의 수 각각 저장
- 양의 수가 늑대의 수보다 많은 경우 양의 개수만 카운트
- 아니라면 늑대의 수만 카운트
- 이미 방문한 곳은 재방문 X
