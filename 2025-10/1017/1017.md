# 📝 알고리즘 문제 풀이 - 2025-10-17

## 문제 📖

- **문제 링크:** [백준 - 윌리암슨수액빨이딱따구리가 정보섬에 올라온 이유](https://www.acmicpc.net/problem/17129)

- **문제 유형:** 그래프

- **난이도:** 실버 1

- **풀이 시간:** ⏳ 35분

## ✍ 접근 방식

- 현재 위치(2)에서 시작하여, 가장 먼저 2, 3, 4(음식) 중 하나에 도달할 때까지의 거리 출력
- 아무 음식도 먹을 수 없으면 NIE 출력
- Python3 시간 초과 이슈로 Pypy로 통과
